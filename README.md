# Week 2 Final Project - Console Application

## Your Mission

Build an interactive console application that solves a problem you care about. This is your chance to bring together everything you've learned in Week 2 and create something you can be proud of.

## What's Included

This repository contains starter code to help you get started:

- `src/main.py` - Basic console application template with main loop
- This README with project guidelines

## Getting Started

1. **Fork and clone this repository**

2. **Review the starter code** in `src/main.py`
   - Shows a basic menu system
   - Demonstrates the main loop pattern
   - Handles user input and exit conditions

3. **Plan your project**
   - What problem will you solve?
   - What features will your MVP include?
   - How will you use file I/O?

4. **Get instructor approval**
   - Write your 2-sentence problem statement
   - Send it via Slack before lunch on Day 1
   - Wait for approval before coding

5. **Start building**
   - Customize the starter code for your project
   - Add additional Python files as needed
   - Focus on your MVP first

## Project Requirements

Your application must:

1. **Be interactive** - Main loop with user input
2. **Use file I/O** - Read from or write to files
3. **Be organized** - Multiple Python files
4. **Solve a specific problem** - Clear purpose and value

See the [Learn lessons](https://learn.galvanize.com) for detailed requirements and resources.

## Project Structure

Start simple and add files as needed:

```
week-2-final-project/
├── src/
│   ├── main.py          # Your main loop (customize this!)
│   ├── functions.py     # Your core functionality
│   └── file_handler.py  # Your file I/O operations
├── data/                # Your data files (if needed)
├── tests/               # Your tests (optional)
└── README.md            # Project documentation
```

## Tips for Success

- **Start small** - Get your MVP working first
- **Test often** - Run your code frequently
- **Commit regularly** - Save your progress to Git
- **Ask for help** - Don't struggle alone for hours
- **Stay focused** - Build something simple that works
